Created a complete website for managing a mountain cottage business. Used HTML, CSS, JavaScript with jQuery library, and Bootstrap framework for the frontend, while PHP was employed for the backend, establishing the connection to the database. 
This project use HTML, CSS, Bootstrap and JavaScript with JQUERY library for everything related to the frontend of the website.
In order to do the backend processes I used PHP and MYSQL in order to create the databases need for this website.
This project also comes with Technical Documentation and User Guide.
